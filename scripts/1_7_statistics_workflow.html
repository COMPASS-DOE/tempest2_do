Purpose

This markdown documents the statistics workflow used for the TEMPEST2 story. The markdown is intended to specifically identify which statistics were used, in what order, and why they were needed. This is resulting from discussions with AMP/Pat/other co-authors about the need for more rigorous testing of statistical assumptions to make sure the tests were using are valid, and the results are trustworthy.

Workflow

This is the statistics workflow for deciding what test to use. All tests are comparing the means of different groups to each other, so the first decision point is based on testing distributions for normality.
